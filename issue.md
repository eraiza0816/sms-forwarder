# 前提条件
実機にインストールは出来るが，adbを使ってのデバッグが現在ユーザが出先なので，出来ない。

## 解消した課題
- **このプロジェクトのアプリは，実機にインスト－ルをして起動すると即座に停止してしまう。これを解消したい。**
    - **解決の経緯:**
        1.  当初、`AppCompat`テーマが適用されていないことが原因で`IllegalStateException`が発生し、アプリがクラッシュしていた。`themes.xml`と`colors.xml`を作成し、`AndroidManifest.xml`でテーマを適用することで解決。
        2.  次に、ビルド時にリソースが見つからないエラーが発生。`Material Components`ライブラリへの依存関係が不足していたため、`build.gradle.kts`に追加して解決。
        3.  ストレージ権限の問題でログが取得できない状況が発生。Androidのバージョンによるストレージ仕様の変更に対応するため、ログの保存場所をアプリ専用の外部ストレージ領域に変更し、権限要求を不要にした。
        4.  ログをアプリUI上に表示するようにし、デバッグを容易にした。
        5.  最終的に、Discord WebhookへのJSONペイロードが無効であることが判明。`JSONObject`を使用してJSONを正しく構築するように修正し、SMS転送が成功するようになった。
